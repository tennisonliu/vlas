Directories
* ./sample_collection: contains all scripts used to collect trigger word samples
* ./sample_augmentation: scripts used to augment the data to create training samples
* ./twd_training: scripts used in training model to detect for trigger words
* ./vlas_system: code base for virtual lab assistant project, execute main.py to run system
* ./documentation: contains documentation for the overall system, documentation for sample collection
	and sample augmentation (Pre-Training Procedure) and trigger word detection training (TWD_training)

NOTE: scripts in /sample_augmentation and /twd_training are optimised for HPC Bracewell
